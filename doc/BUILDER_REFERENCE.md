## Keyboard Shortcuts

<a id="builder-shortcuts-presentation"></a>

### Presentation Actions

| Key | Action |
|-----|--------|
| `Ctrl+S` | Save the current presentation |
| `F5` | Show full presentation (presentation must be saved first) |
| `Shift+F5` | Show presentation starting from the currently selected slide |

---

<a id="builder-shortcuts-nav"></a>

### Slide Navigation

These shortcuts move the active selection in the slide list. The **Ctrl** variants work everywhere including while editing text; the plain arrow keys work only when no text field is focused.

| Key | Action |
|-----|--------|
| `ArrowLeft` / `ArrowRight` | Previous / next column |
| `ArrowUp` / `ArrowDown` | Previous / next slide in the current column |
| `Ctrl+ArrowLeft` / `Ctrl+ArrowRight` | Previous / next column |
| `Ctrl+ArrowUp` / `Ctrl+ArrowDown` | Previous / next slide in the current column |
| `Ctrl+PageUp` | Jump to the first slide in the current column |
| `Ctrl+PageDown` | Jump to the last slide in the current column |
| `Ctrl+Home` | Jump to the very first slide (column 1, slide 1) |
| `Ctrl+End` | Jump to the very last slide (last column, last slide) |

---

<a id="builder-shortcuts-editing"></a>

### Slide Editing

| Key | Action |
|-----|--------|
| `Ctrl+M` | Add a new slide after the current slide |
| `Ctrl+D` | Duplicate the current slide |
| `Ctrl+H` | Hide or Unhide the current slide |
| `Ctrl+L` | Create a New Column |
| `Ctrl+Enter` | Break the current slide at the cursor position (split into two slides) |
| `Shift+Enter` | Insert a markdown line break (`\`) |

<a id="builder-shortcuts-plugins"></a>

### Content Creators (Plugin Shortcuts)

These shortcuts open content-insertion panels. They are only available when the corresponding plugin is enabled.

| Key | Action |
|-----|--------|
| `Ctrl+B` | Open the Bible Text creator |
| `Ctrl+Y` | Open the Hymnary song search |
| `Ctrl+W` | Open the Adventist Hymns search |

<a id="builder-shortcuts-sorter"></a>

### Slide Sorter

The Slide Sorter is a full-screen drag-and-drop view for reordering slides across columns.

| Key | Context | Action |
|-----|---------|--------|
| `Escape` | Builder (Slide Sorter closed) | Open the Slide Sorter |
| `Escape` | Slide Sorter | Close the Slide Sorter and return to the builder |
| `ArrowLeft` / `ArrowRight` | Slide Sorter | Previous / next column |
| `ArrowUp` / `ArrowDown` | Slide Sorter | Previous / next slide in the current column |
| `Home` | Slide Sorter | Jump to the first slide (column 1, slide 1) |
| `End` | Slide Sorter | Jump to the last slide (last column, last slide) |
| `PageUp` | Slide Sorter | Jump to the first slide in the current column |
| `PageDown` | Slide Sorter | Jump to the last slide in the current column |
