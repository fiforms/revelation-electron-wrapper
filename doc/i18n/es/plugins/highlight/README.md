## Resumen

El plugin Highlight habilita temas de resaltado de sintaxis para bloques de código.

---

<a id="highlight-what-it-adds"></a>
## Qué agrega

- Integración de resaltado de sintaxis del lado del cliente
- Temas de hoja de estilo highlight seleccionables

---

<a id="highlight-configuration"></a>
## Configuración

- `stylesheet`: nombre de archivo del tema CSS (selector desde archivos `.min.css` incluidos)
