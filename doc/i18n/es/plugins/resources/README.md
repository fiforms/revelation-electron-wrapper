## Resumen

El plugin Resources proporciona una entrada de página Resources en la UI de plugins.

<a id="resources-what-it-adds"></a>
## Qué agrega

- Un botón de plugin que abre `index.html` para explorar recursos
- Registro de plugin expuesto al navegador para integración de UI

Este plugin es intencionalmente liviano y está enfocado principalmente en enrutamiento de UI.
