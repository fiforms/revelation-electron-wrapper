## Resumen

El plugin Hymnary busca en Hymnary.org e inserta letras de himnos de dominio público como diapositivas markdown.

---

<a id="hymnary-what-it-adds"></a>
## Qué agrega

- Diálogo de búsqueda de himnos con soporte de idioma/filtros
- Obtención y análisis de letras desde páginas de himnos
- Generación de diapositivas con reconocimiento de estrofa/estribillo
- Opción de agregar al archivo activo de presentación

---

<a id="hymnary-how-it-works"></a>
## Cómo funciona

El plugin consulta exportaciones de Hymnary (CSV) para búsqueda, obtiene páginas de himnos seleccionadas, analiza letras y metadatos, y produce markdown separado por diapositivas.
