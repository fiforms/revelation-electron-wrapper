## Overview

The Resources plugin provides a Resources page entry in the plugin UI.

<a id="resources-what-it-adds"></a>
## What It Adds

- A plugin button that opens `index.html` for resource browsing
- Browser-exposed plugin registration for UI integration

This plugin is intentionally lightweight and primarily UI-routing focused.
