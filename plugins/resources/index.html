About REVELation Snapshot Presenter

REVELation Snapshot Presenter is a modular presentation builder for pastors, teachers, and creators who want fast, powerful workflows using Reveal.js.

It integrates tightly with Markdown, a full media library, Bible text tools, Adventist Hymns, the Virtual Bible Snapshot image system, and automated import/export pipelines.

This tool is developed together with the Virtual Bible Snapshot Project.

Desktop Editors

Any text editor or Markdown-capable editor works with REVELation, but it should support seamlessly loading and savings files from your local filesystem (so web-based editors are not ideal).
