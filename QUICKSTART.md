# Welcome to REVELation Snapshot Presenter

REVELation Snapshot Presenter helps you create and present beautiful, media-rich slides without needing web development skills.

It combines a friendly desktop app, markdown-based authoring, and a flexible Reveal.js presentation engine.

## What You Can Do

* Create presentations with markdown and reusable macros
* Add images, video, and audio quickly
* Manage presentations and media from a desktop GUI
* Export for handouts, offline sharing, and live presenting
* Extend behavior with plugins

---

## Typical Workflow

1. Install and open REVELation Snapshot Presenter.
2. Create a presentation from the app.
3. Add media to your library.
4. Write slides in markdown.
5. Present live or export for sharing.
